# Projeto MMTour automação

### Como executar
Para começar a usar, siga estes passos:

1. **Instale o pipx**:
   ```bash
   pip install pipx
   ```
2. **Intale o Poetry**:
   ```bash
   pipx install poetry
   ```
3. **Inicie o Poetry e instale as bibliotecas**:
   ```bash
   poetry init
   ```
4. **Rode a aplicação**:
   ```bash
   poetry run pyhton src/app.py
   ```
